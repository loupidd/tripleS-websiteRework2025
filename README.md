# SSSolusindo Website — UI Rebuild & Rework

This project is the rebuilt version of the SSSolusindo website, focusing on UI improvements and user experience.

###  What's New
- **UI & Structure** — Refined alignment, layout structure, and consistent UI elements.
- **Navigation** — Improved navbar flow for easier browsing.
- **Visual Design** — Adopted subtle details inspired by Apple's new glass-like design.
- **Contact Form** — Reworked from a simple WhatsApp redirect into a fully formatted Google Sheets + WhatsApp integration.

###  Tech Stack
- **Frontend:** HTML, Tailwind CSS, Vanilla JS  
- **Animation & UX:** (e.g., Flowbite, AOS/GSAP if applied)  
- **Integration:** Google Apps Script for form submission → Google Sheets
